# Introduction #

The project started in April 2012 by Robotic Masterstudents  with the idea to build a real robot for a realistic application. After a creative conceptPhase we decided to build a team of small wheeled robots for rescue applications.

**Project Goal:**
We started this project to learn creating a robot from the scratch. Starting by defining the Task our robot should fullfill, define specific requirements and get the aproppriate hardware...

**Team**
The team consists of 4 students of the Robotic Master and one Automotive Master student.
